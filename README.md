# Robotics Ridge - Train control Node app

## Setup instructions
Install with:<br/>
`npm install`

Edit and configure the `.env` file:
```
domain='https://test.salesforce.com'
callbackUrl='http://localhost:5000/'
consumerKey=''
consumerSecret=''
apiVersion='v45.0'

sfdcUsername=''
sfdcPassword=''

isMockTrain=false
```

Build with:<br/>
`npm run build`

Start with:<br/>
`npm start`
